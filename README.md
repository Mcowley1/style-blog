# style-blog

## Description
This app allows users to log in, sign up, view, add, update and remove comments on a blog.

## Tools
* JavaScript
* CSS
* Handlebars
* partials (From Handlebars)
* Node.js
* Express
* Express-Handlebars
* MySQL2
* Sequelize
* Dotenv
* Heroku
